Estes codigos son ejemplos de uso de la jerarquia de clases
desarrollada para implementar metodos de descomposicion de
dominio para la resolucion de ecuaciones diferenciales
parciales, entre otros.

Para compilar, ejecutar y limpiar los archivos temporales 
generados durante la compilacion se desarrollo un makefile

Para compilar
 $ make
 
Para ejecutar
 $ make run
 
Para borrar los archivos temporales
 $ make clean
